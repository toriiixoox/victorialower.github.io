The Start of My Passion
Multi-Page Layout


Description:
One column fixed layout with four pages including a home page, a gallery with 6 separate image pages, a list of items with custom bulletpoints, and an about us. Also contains sited photos throughout the website, a custom logo, and subtle background.

Challenges:
This was my first big project in my class. I was given a list of criteria that I had to execute in the code. Figuring out the basics of coding was a challenge that I had a fun time tackling. For example, making the nav inline instead of as bulletpoints and how to position text under images.

Things I'm Proud Of:
Was able to figure out solutions to problems that arose while coding. For example, positioned citation of photos in a visually pleasing way that still allowed for credit to be given, made title visible by adding a stroke weight and adding a purple layer to the photo.

What It's Taught Me:
I have a true passion for UX/UI. I had never coded websites before, but this project showed me how much fun it is figuring out how to make the website look exactly like I want it to look. Making the website function, while still creating a visually pleasing design was very fulfilling.
